# Personalized Workout & Diet Planner with AI

**Description:**  
Generates **customized workout routines and diet plans** for students based on age, weight, height, fitness goal, fitness level, diet preference, and available time. Primarily **rule-based**, with optional **AI enhancement using Scikit-learn KNN** for smarter suggestions.

---

## Features

- Personalized workout plan:
  - Warm-up, main exercises, core, cool-down
  - Time-based selection: 15 / 30 / 45 / 60 mins
  - Fitness goals: Weight Loss / Muscle Gain
  - Fitness levels: Beginner / Intermediate / Advanced
- Personalized diet plan:
  - Breakfast, lunch, dinner
  - Diet preference: Vegetarian / Non-Vegetarian / Both
- BMI calculation with status
- Rule-based system fully functional
- Optional AI enhancement for varied exercise/diet recommendations
- Responsive, screenshot-friendly UI

---

## Tech Stack

- **Frontend:** HTML, CSS (responsive, icon-enhanced)
- **Backend:** Python, Flask
- **AI / ML:** Scikit-learn KNN (optional)
- **Libraries:** NumPy

---

## Setup

1. Clone repo: `git clone <repo-url>`
2. Create virtual env: `python -m venv venv`
3. Activate env:
   - Windows PowerShell: `venv\Scripts\Activate.ps1`
   - Windows CMD: `venv\Scripts\activate.bat`
   - Linux/Mac: `source venv/bin/activate`
4. Install dependencies: `pip install -r requirements.txt`
5. Run app: `python app.py`
6. Open browser: `http://127.0.0.1:5000`
